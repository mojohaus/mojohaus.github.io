Codehaus support

Codehaus offers the following systems to support open source development.

Application URL Description
Xircles http://xircles.codehaus.org Maintain your personal settings (except Jira credentials, which is not part of the LDAP)
Jira http://jira.codehaus.org/ Issue tracker
Bamboo http://bamboo.ci.codehaus.org/ Continuous Integration Server
Fisheye http://fisheye.codehaus.org/ Source Repository Viewer
Confluence http://docs.codehaus.org/ Wiki
Sonar http://sonar.ci.codehaus.org/ Code Quality Management System
IRC http://irc.codehaus.org/ IRC Webclient for Codehaus Internet Relay Chat

There's a personal webspace available for every committer. Due to HAUS-2187 it's not generated automaticly, instead you have to ask for it.
