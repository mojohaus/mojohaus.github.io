Mail Template for a Release Vote

This templates contains most of the data developers expect to do validate a vote.
Replace all UPPER_CASE_TERMS with its equivalent value. You're free to add or skip sections, just see this as a good starting point.

Subject [VOTE] Release PROJECT_NAME version PROJECT_VERSION
To dev@mojo.codehaus.org
Hi,

I'd like to release version PROJECT_VERSION of the PROJECT_NAME

PROJECT_DESCRIPTION

We solved NR_OF_SOLVED_ISSUES issues:
http://jira.codehaus.org/secure/ReleaseNote.jspa?PROJECTID_AND_VERSION

There are still a couple of issues left in JIRA:
http://jira.codehaus.org/secure/IssueNavigator.jspa?A_LOT_OF_PARAMETERS

Staging Repositories:
General:  https://nexus.codehaus.org/content/groups/staging/
Exclusive: https://nexus.codehaus.org/content/repositories/SOME_SPECIFIC_STAGING_VERSION

(Staging) Site:
PROJECT_URL AND/OR STAGING_SITE_URL

SCM Tag:
PROJECT_SCM_URL

ADDITIONAL_INFORMATION

 [ ] +1
 [ ] +0
 [ ] -1

The vote is open for 72 hours and will succeed by lazy consensus.



Regards / Cheers / 

YOUR_NAME
