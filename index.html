Welcome to the MojoHaus Project (previously known as Mojo@Codehaus)

The MojoHaus project is a collection of plugins for Apache Maven.

Releases of the plugins are available from the Maven Central repository. Snapshots are available on request through the development list or by emailing the maintainer. Each plugin has a list of people involved in creating or supporting it.

Plugins

For information on the plugins hosted at the MojoHaus project see the plugins list. Detailed information and instructions for each plugin can be found on its Maven site.

Getting Support

If you have problems or concerns with plugins in the MojoHaus project please contact the author, ask on the regular Maven Users mailing list. There are also instructions on how to report an issue should you run into one.

Development

If you would like to contribute to the project, either through fixing an existing plugin or by donating a new plugin, contact us on the development list.
